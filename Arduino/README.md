You need: 7 MPU6050, 1 PCA9548A and Arduino Uno (nano)
Maximum IMU count is 8, but you can use more if add 2 more PCA9548A

# Dependencies
Libraries: I2Cdev, MPU6050_6Axis_MotionApps20

# Output format
Output format is:
`IMU_N acc_x acc_y acc_z roll pitch yaw`
